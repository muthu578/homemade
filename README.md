# Pranavika’s Creations – Boutique, Bakery & Homemade

**Fashion | Cakes | Homemade Delights**

## 📂 Project Structure
- `/client` - Next.js 15+ Frontend (React, Tailwind, Recharts)
- `/server` - Node.js + Express Backend (Planned)
- `/shared` - Shared types and constants (Planned)

## 🎨 Branding
- **Logo**: Custom monogram "P" with whisk, dress silhouette, and needle.
- **Colors**: Blush Pink, Cream, Cocoa Brown.

## 🚀 Getting Started
### Prerequisites
- Node.js 18+
- npm / yarn

### Development
```bash
# Frontend
cd client
npm install
npm run dev
```
